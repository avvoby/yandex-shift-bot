# Telegram-бот для проекта мерчендайзеров в Яндекс Смене

Это Telegram-бот для сопровождения исполнителей проекта по мерчендайзингу.

## Что умеет бот

### Для исполнителей
- регистрация по ФИО и номеру телефона;
- сохранение данных в Google Sheets;
- главное меню с полезными разделами;
- FAQ из Google Sheets;
- получение инструкций и ссылок;
- отправка вопроса в поддержку;
- ответы на служебный опрос по смене:
  - Да, буду на задании
  - Нет, нужно отменить задание
  - Есть вопрос

### Для администраторов
- админ-меню внутри Telegram;
- массовые рассылки;
- рассылки по списку Telegram ID;
- запуск опроса по смене по выборке;
- редактирование текста опроса перед отправкой;
- принудительное обновление контента из Google Sheets;
- журнал рассылок и ответов пользователей.

## Технологии
- Python 3.11+
- aiogram 3
- Google Sheets API
- long polling
- Yandex Cloud VM

## Общая логика
- тексты и FAQ лежат в Google Sheets;
- бот кэширует их на несколько минут;
- админ может нажать "Обновить контент сейчас";
- вопросы пользователей пересылаются в отдельный чат поддержки;
- ответы на опросы по смене сохраняются в Google Sheets.

## Важно
Код проекта снабжен комментариями, чтобы его было проще поддерживать и дорабатывать.
